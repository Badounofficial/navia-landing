## How to Add New Sources

When writing a new article:
1. Research from authority sources (NIH/PMC, WHO, Mayo Clinic, Harvard Health, Johns Hopkins, Cleveland Clinic, ACOG, Lancet, JAMA)
2. Record each source in the appropriate section above
3. Note the exact data point used and which article(s) reference it
4. Weave the data into prose naturally (per brand guidelines)
5. Update this file before publishing

## Source Quality Tiers

- **Tier 1 (Gold):** NIH/PMC, WHO, Lancet, JAMA, NEJM, Cochrane Reviews
- **Tier 2 (Silver):** Mayo Clinic, Harvard Health, Johns Hopkins, Cleveland Clinic, ACOG, institutional .edu
- **Tier 3 (Bronze):** Healthline, WebMD, Psychology Today, established health journalism
- **Avoid:** Blogs, affiliate content, unattributed statistics, Wikipedia as primary source
