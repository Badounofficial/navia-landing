[1-2 paragraphs. The sentence that lingers.]

## Checklist Before Submission

- [ ] Character count within format range
- [ ] All 3 internal links present (pillar, cross-cluster, sibling)
- [ ] At least 1 external authority link
- [ ] Statistics verified against references.md
- [ ] No statistics repeated from existing articles (check improvement-log.md)
- [ ] Opening structure differs from all existing articles
- [ ] No em-dashes or en-dashes
- [ ] No banned phrases
- [ ] No competitor mentions
- [ ] No functional language
- [ ] Scoring rubric applied, total > 90/100
- [ ] Improvement log updated with new statistics, themes, phrases
- [ ] References.md updated with new sources
