# Ozaia Visual Universe: Midjourney Prompts

*Organized by category. Copy each prompt directly into Midjourney. All prompts use v6.1 syntax.*

**Visual rules (apply to everything):**
- No identifiable faces. Silhouettes, partial views, hands, shoulders, back of head only.
- No text on images (add text later in Canva).
- Night palette: indigo (#2B2940), warm gold (#E8C58F), soft rose (#E8A89D), deep purple (#4A3D6B).
- Day palette: lavender (#F4EFF5), muted purple (#EFEAF6), dusty rose (#C97A8E).
- Cinematic grain, shallow depth of field, intimate framing.
- Moon as recurring motif. Never full and perfect. Always a crescent, slightly imperfect, organic.

## Production Notes

### Midjourney Setup
- Use `/settings` and select **v6.1** model
- Stylize (`--s`) between 750-950 for cinematic quality
- Always check results at 2x upscale before using
- If a result is close but not perfect, use `--seed [number]` with variations

### Post-Production (Canva)
- Add Ozaia wordmark (Fraunces font, moon dot on the i) in top right corner, small and discreet
- For social posts, add text using Inter 300 for body, Fraunces 400 for headlines
- Apply a subtle film grain overlay at 5-10% opacity for consistency
- Never add more than one line of text on an image. Let the visual breathe.
